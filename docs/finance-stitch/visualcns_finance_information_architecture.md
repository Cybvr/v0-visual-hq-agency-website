# Visualcns Finance - Information Architecture

## 1. Public Marketing Site (Sitemap)
- **Home**: Platform overview, "Complete PE Operating System" value prop.
- **Solutions**:
    - Sourcing & CRM
    - QofE & AI Modeling (Featured)
    - Portfolio Monitoring
    - LP Relations
- **Case Studies**: Quantitative success stories.
- **Resources**: Blog, Whitepapers, Documentation.
- **Company**: About, Careers, Contact.
- **Legal**: Privacy, Terms, Security.

## 2. Web Application (App Map)
- **Global Navigation**:
    - Search (Targets, Models, Reports)
    - Notifications
    - User Profile / Settings
- **Dashboard (Home)**: Portfolio health snapshot, active deal pipeline cards, recent report activity.
- **Deal Pipeline**:
    - Pipeline View (Kanban/List)
    - Target Detail Pages
    - Sourcing Analytics
- **Analysis Hub (The Core Engine)**:
    - **Step 1: Business Info Entry**: Deal parameters, reporting framework.
    - **Step 2: Document Intake**: Secure VDR integration, manual uploads, AI status tracker.
    - **Step 3: Analysis Engine**: Trial balance mapping, adjustment flagging, EBITDA bridge.
    - **Step 4: Report Output**: Branded QofE reports, export center.
- **Financial Modeling**:
    - LBO/DCF Model Library
    - Valuation Templates
    - Excel Integration Settings
- **Portfolio Monitoring**:
    - Holding Company List
    - KPI Performance Tracking
    - Value Creation Initiative Tracking
- **LP Reporting Portal**:
    - Fund Performance Overviews
    - Document Distribution Center (K-1s, Quarterly Decks)
    - Secure LP Messaging

## 3. Data & AI Layer (Underlying Logic)
- **Document Lens**: OCR and data extraction engine.
- **Adjustment Engine**: AI-detected financial anomalies.
- **Benchmarking Engine**: (Planned) Industry data comparison.
- **Audit Trail**: Version control and compliance logging for all adjustments.
