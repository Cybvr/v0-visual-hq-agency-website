# Visualcns Finance - Product Design Specification (MVP)

## Project Overview
AI-powered web application automating Quality of Earnings (QofE) reporting for private equity.

## Target Audience
Investment professionals, private equity analysts, and finance-oriented users at smaller firms.

## Core User Flow
1. **Landing Page**: Professional introduction to the AI-powered automation value prop.
2. **Onboarding / Business Entry**: Inputting company details (Revenue, EBITDA, Industry).
3. **Document Intake**: Drag-and-drop for Excel and PDF files with AI status tracking.
4. **Standardized Report Output**: The final "Branded Report" showing analyzed earnings, adjustments, and AI insights.

## Design Constraints
- **Name**: Visualcns Finance
- **Colors**: Blue and White
- **Fonts**: EB Garamond (Landing Headings), Inter (App & Body)
- **Device**: Desktop
